## 无问西东

**等你们长大，**

**你们因绿芽冒出土地而喜悦，**

**会对出升的朝阳欢呼跳跃，**

**也会给别人善意和温暖，**

**但是却会在赞美别的生命的同时，**

**常常 甚至永远地忘了自己的珍贵**

**愿你在被打击时，记起你的珍贵，抵抗恶意**，

**愿你在迷茫时，坚信你的珍贵，**

**爱你所爱 行你所行，听从你心 无问西东**
